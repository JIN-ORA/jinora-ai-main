# JINORA YouTube Audio Downloader

## 소개
JINORA의 첫 번째 시범 프로젝트입니다.

이 프로젝트는 유튜브 미디어에서 음원을 추출하여 다운로드할 수 있도록 만든 도구입니다.  
아이들이 사용하는 음원펜, 예를 들어 세이펜과 같은 기기에 필요한 음원을 넣어 사용할 수 있도록 돕는 것을 목표로 합니다.

육아를 하다 보면 아이가 좋아하는 노래, 영어 음원, 동화, 학습 콘텐츠 등을 반복해서 들려주고 싶을 때가 많습니다.  
JINORA는 이런 상황에서 부모들이 조금 더 편하게 필요한 음원을 준비할 수 있도록, 실생활에 도움이 되는 기능을 하나씩 추가해 나가려 합니다.

누구나 다운로드해서 사용할 수 있고, 필요한 기능은 사용자 요청을 받아 지속적으로 개선할 예정입니다.

## 주요 기능
- 유튜브 미디어에서 음원 추출
- 추출한 음원 파일 다운로드
- 음원펜, 세이펜 등 외부 기기에 넣어 사용할 수 있는 파일 생성
- 육아에 필요한 음원 관리 기능 확장 예정
- 사용자 요청 기반 기능 개선

## 사용 방법

1. 프로그램 다운로드

프로젝트 파일을 다운로드합니다.

2. 프로그램 실행

다운로드한 프로그램을 실행합니다.

3. 유튜브 링크 입력

음원으로 추출하고 싶은 유튜브 링크를 입력합니다.

4. 음원 다운로드

추출된 음원 파일을 다운로드합니다.

5. 음원펜에 파일 복사

다운로드한 음원 파일을 세이펜, 음원펜 또는 사용하는 기기에 복사하여 사용합니다.

## 앞으로 추가할 기능
- 음원 파일명 자동 정리
- 여러 개의 유튜브 링크 일괄 다운로드
- 아이 연령별 추천 음원 관리
- 세이펜용 폴더 구조 자동 생성
- 다운로드 이력 관리
- 사용자가 요청한 육아 편의 기능 추가

## 주의사항
이 프로그램은 개인적인 학습 및 가정 내 사용을 돕기 위한 목적으로 제공됩니다.

유튜브 콘텐츠의 저작권은 원 저작자에게 있으며, 다운로드한 음원은 저작권 정책을 준수하여 개인적인 용도로만 사용해 주세요.  
상업적 이용, 무단 배포, 저작권 침해에 해당하는 사용은 금지됩니다.
